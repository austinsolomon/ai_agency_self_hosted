# Skills Directory

This directory contains skill libraries and capabilities for the AI Agency platform.

## n8n-skills

Located in `./n8n-skills/`

A comprehensive Claude Code plugin providing AI-powered skills for working with n8n workflows.

**Features:**
- Pre-built n8n workflow skills
- Node evaluation and testing capabilities
- Integration with Claude Code
- Workflow best practices and patterns

**Source:** https://github.com/czlonkowski/n8n-skills

**Documentation:** See `./n8n-skills/README.md` for detailed usage instructions.
